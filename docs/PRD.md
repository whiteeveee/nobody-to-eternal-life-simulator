# Nobody to Eternal Life Simulator（人生模拟器：凡人→修仙）PRD v0.2

## 1. 一句话定位
一款文字驱动的人生模拟器：玩家选择出身与初始配置，角色在“随机人生小事件 + 十年/二十年关键选择分支”的驱动下成长或早逝；若活到 100 岁并满足入道条件，可进入修仙阶段，最终渡劫成仙（或失败结局）。每局结算获得局外累计属性点，用于下局开局自由加点（有上限）。

## 2. 核心体验目标
- 一局时长不固定：可能 0 岁暴毙，也可能一路到 100 岁再修仙通关
- 自动模拟为主：随机小事件快速推进人生
- 关键节点玩家决策：每 10 年或 20 年出现一次重要分支，显著改变属性与后续走向
- 二阶段爽感兑现：凡人期的选择/积累，会在修仙期转化为优势或负债
- 局外成长：每局结束给少量局外点，下局可更强（有上限，防止膨胀）

---

## 3. 重要名词与数值

### 3.1 时间轴
- **年龄（Age）**：时间轴/进度，不作为可分配属性。
- 凡人期从 0 岁推进到 100 岁（或中途死亡）。
- 每 10 年为一个“结算段”（0–10、10–20…90–100）。

### 3.2 凡人期核心属性（初始由出身决定）
- **体魄（Body）**：疾病、意外、寿命相关检定；低体魄更容易提前结束
- **智慧（Mind）**：学习、考试、策略、悟性相关事件效果
- **仙缘（Fate）**：奇遇、贵人、入道机会、修仙顺利程度
- **钱财（Wealth）**：资源与避灾能力；过低可能触发破产/饥寒等风险事件

> 说明：年龄是时间轴；体魄/智慧/仙缘/钱财为凡人期核心属性。

### 3.3 修仙期新增属性（进入修仙后启用）
- **灵根（Root）**：修炼效率与境界上限
- **法器（Gear）**：斗法与雷劫生存能力
- **修为（Cultivation）**：境界推进进度条

---

## 4. 一局流程（Core Loop）

### 4.1 开局
1) 选择出身（决定初始体魄/智慧/仙缘/钱财，以及可选物品池）
2) 选择初始物品（可包含“入道信物”与“凡间物品”）
3) 分配局外累计属性点（仅对体魄/智慧/仙缘自由加点；有上限）

### 4.2 凡人期推进（自动模拟为主）
- 以“十年段”为主推进：每到一个十年段结尾触发一次结算。
- **十年段内穿插随机小事件**（不受玩家控制）：
  - 随机隔几年出现一句文案并改变属性，例如：
    - 6 岁：被家里人带去少年宫参加活动，智慧 +2
    - 8 岁：抢同桌零食被叫家长，仙缘 -1
    - 9 岁：学老爸在厕所抽烟呛到，体魄 -2
- **十年/二十年关键分支事件**（玩家选择）：
  - 例如：升学、择业、婚恋、重大机遇、关键风险
  - 选项会显著改变属性（并可影响后续事件倾向/权重）

### 4.3 100 岁结算与入道判定
- 若角色提前死亡：直接进入结局结算与局外奖励
- 若角色活到 100 岁：
  - 若满足入道条件（例如：持有入道信物 + 仙缘达到门槛 等）→ 进入修仙阶段
  - 否则：凡人寿终正寝结局

---

## 5. 事件系统设计

### 5.1 事件类型
- **随机小事件（自动）**：短文案 + 小幅属性变化；主要用于丰富人生与形成“命运感”
- **关键分支事件（玩家选择）**：每 10/20 年触发；大幅改变属性/路线/后续权重
- **风险事件（自动）**：当体魄/钱财过低等条件满足时更容易触发（用于制造提前结束）

### 5.2 十年段推荐结构
每个十年段：
1) 若干条随机小事件（0–3 条，隔随机几年触发）
2) 末尾触发“十年结算”（可能包含关键分支事件）
3) 判定是否进入下一十年段（死亡/继续）

---

## 6. 物品系统

### 6.1 物品分类
- **入道信物**：影响能否在 100 岁开启修仙；不同信物会影响修仙期走向/加成/风险
- **凡间物品**：在凡人期发挥作用（保命、变现、增属性、触发事件）

### 6.2 示例规则（后续扩内容）
- 例：凡间物品“金条”
  - 当钱财低于阈值时，可能触发“卖掉金条”事件 → 钱财回升/避免提前结束
- 入道信物在创建模拟时可选（不同出身可选列表不同），也可在随机事件中以低概率获得（后续补充）

---

## 7. 修仙阶段（认可的设计）
境界顺序：
炼气 → 筑基 → 金丹 → 元婴 → 化神 → 雷劫 → 成仙/失败

每个境界建议结构：
1) 修炼事件（涨修为）
2) 资源/斗法事件（拿法器、灵石等）
3) 突破检定（成功进阶；失败可能掉境/受伤/直接结束）

化神后必触发雷劫：
- 成功：成仙通关
- 失败：结局结束

---

## 8. 局外累计（Meta Progression）

### 8.1 奖励形态
- **局外累计 = 属性点（Meta Points）**
- 下局开局可分配到：体魄/智慧/仙缘（不直接加钱财）

### 8.2 上限与保守发放原则
- 单局结算得到的局外点要“保守”
- **总可用的额外局外点设置上限：20 点**
  - 达到上限后仍可继续游玩，但不再增加局外点（或转为解锁/成就，后续再定）

> 说明：具体发放公式待后续数值平衡阶段确定；当前目标是保证成长感但避免膨胀。

## 9. MVP（第一版可玩范围建议）
- 出身：3 个
- 入道信物：至少 3 个（各自影响修仙走向）
- 凡人随机小事件：30 条
- 十年/二十年关键分支事件：10 条
- 修仙事件：每境界 3–6 条
- 结局：至少 8 个（含早夭、寿终、入道失败、渡劫失败、成仙通关）

## 10.修仙数值体系 v0.3（不×10版）

A. 修仙阶段属性（0–1000），凡人属性只负责“转译入道初值”

凡人阶段（0–100）：
    •    魅力 M、财富 W、体魄 B、智力 I、幸运 L

修仙阶段（0–1000）：
    •    仙缘 X（由 M 转译）
    •    灵石 S（由 W 转译，作为货币/资源）
    •    灵根 R（由 B 为主转译，代表潜力）
    •    神识 N（由 I 转译，代表学习/理解/洞察）
    •    幸运 L（不变，仍用 0–100；它只影响“随机事件倾向”，不影响玩家选择）

修仙过程属性（0–1000）：
    •    修为 P（境界进度条）
    •    法器 A（装备/工具/背书/关系牌）
    •    债务/心魔 D（资本修仙味道建议用“债务/违约压力”来表现心魔：越高越容易翻车）

这套非常贴合你要的赛博修仙基准：修炼资源金融化、法力贷、免息期、租灵根、打药提升等。 ￼

⸻

B. 入道初值转译（关键：不×10，初值只到 60–240 左右）

入道（100岁触发）时，把凡人 0–100 转译为修仙初值（0–1000）：

推荐转译公式（简单、好调参、不会爆炸）
（rand 表示随机浮动，制造“同出身也有差异”）

    •    X0（仙缘） = clamp( 50 + 1.2*M + rand(-25,+25), 0, 1000 )
    •    S0（灵石） = clamp( 60 + 1.5*W + rand(-35,+35), 0, 1000 )
    •    R0（灵根） = clamp( 40 + 1.6B + 0.4I + rand(-40,+40), 0, 1000 )
    •    N0（神识） = clamp( 50 + 1.3*I + rand(-30,+30), 0, 1000 )
    •    L（幸运） 保持 0–100，不转译
    •    P0（修为）= 0（刚入道从 0 起步）
    •    A0（法器） 默认 30–80（由“入道信物/局外点/出身道具”给加成）
    •    D0（债务/心魔） 默认 0–120（天谴之人可更高，比如 80–200，符合“出生欠债、催收威胁”的基调）

这样做的结果是：
    •    入道初值“看起来已经比凡人牛很多”（进入 0–1000 体系）
    •    但远没到离谱，也不会一上来就把“平均 200 突破筑基”这种门槛踩穿。

⸻

C. 境界、寿命上限与“卡关死亡”

你提的寿命机制非常适合肉鸽：
    •    炼气：年龄上限 200（200 内没突破 → 结局）
    •    筑基：350
    •    金丹：550
    •    元婴：800
    •    化神：1000
    •    1000 岁：强制结局
    •    成仙（渡劫成功）= 真通关
    •    没成仙但活到 1000（混成千年老怪/债务缠身/摆烂躺平）= “另类通关/隐藏结局”（你说“反而通关”，我建议就做成这个）

⸻

D. 突破判定：用“平均值门槛”+“进度条门槛”

你说“平均到 200 突破筑基…类推”，我们就按这个来落地，规则非常直觉：

每个境界突破需要同时满足：
    1.    P ≥ P门槛（修为进度条达标）
    2.    avg ≥ avg门槛（核心能力平均值达标）
    •    avg = (R + N + X + A) / 4  （不把灵石算进 avg，避免“有钱直接突破”太粗暴；钱应该走“买药/买法器/租灵根→影响 A 或 P 增长”）

建议门槛（可调）：
    •    炼气 → 筑基：P ≥ 200 且 avg ≥ 200（且年龄 < 200）
    •    筑基 → 金丹：P ≥ 400 且 avg ≥ 400
    •    金丹 → 元婴：P ≥ 600 且 avg ≥ 600
    •    元婴 → 化神：P ≥ 800 且 avg ≥ 800
    •    化神 → 雷劫：P ≥ 950 且 avg ≥ 900（再加 D 必须低于某个值，比如 D ≤ 700）

失败后果（资本修仙味）：
    •    失败：P -50，D +80（并触发“法力贷逾期/挖灵根抵债/强制打工”事件链）
    •    失败次数达到上限可直接结局（或被迫“卖灵根/签卖身契”续命）
